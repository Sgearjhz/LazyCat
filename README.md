# LazyCat前端（早期整版)

此为阉割版本…因为完整版本被卖家加钱卖TV4...所以停止后续的开源了

可以采用XCode安装应用到AppleTV4或者模拟器中，下载工程后改一下Bundle Idendifiler和Signing名运行即可

如果没有mac或xcode可以采用TestFlight的方式安装，请到 https://fuzhuo.github.io 中回复并附上邮箱地址，会发送测试邀请
